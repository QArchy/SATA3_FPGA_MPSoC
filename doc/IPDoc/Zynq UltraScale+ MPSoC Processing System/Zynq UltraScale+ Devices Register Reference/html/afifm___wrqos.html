DescriptionQoS Write Channel Register

Sets the static QoS value to be used for the write channel. If APB register field, 'FabricQoSEn' is 0 , this static QoS value will be applied to all write commands enqueued into the WrCmdQ. If 'FabricQoSEn' is 1 ), this static QoS field will be ignored.
