SIZE28:2woWrite-only0x0Specifies the number of 4-byte words the QSPI will read from memory
Size is word aligned, so this field is only 27-bits. (2 lsbs are 0)
The value in this register signifies the number of words to be read from memory.
This is a static signal - should not be changed while controller is actively transmitting/receiving data to/from Flash device
