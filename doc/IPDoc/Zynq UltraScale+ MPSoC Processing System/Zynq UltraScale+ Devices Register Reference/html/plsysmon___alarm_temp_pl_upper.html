ALARM_TEMP_PL_UPPER (PLSYSMON) Register Description

Register NameALARM_TEMP_PL_UPPER
Relative Address0x0000000140
Absolute Address 0x00FFA50D40 (AMS_PL_SYSMON)
Width16
TyperwNormal read/write
Reset Value0x00000000
DescriptionPL Upper temperature Alarm Threshold.

Thermal diode is located in the PL SysMon unit, toward the center of the PL die area. Refer to the ALARM_TEMP_PL_LOWER register and UG580 for more information.
