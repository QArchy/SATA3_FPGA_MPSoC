When this event is set, it means that the ramp time is completed (GADPCTL.RTIM has reached its terminal value of 0x7FF). This is a debug feature that allows software to read the ramp time after each cycle.

Reserved24:16roRead-only0x0Reserved
RTIM15:0roRead-only0RAMP TIME:
These bits capture the latest time it took for Vbus to ramp from VADP_SINK to VADP_PRB. The bits are defined in units of 32 kHz clock cycles as follows:
- 0x000: 1 cycles
- 0x001: 2 cycles
- 0x002: 3 cycles and so on till,
- 0xFFFF: 65536 cycles
The maximum time of 65536 cycles corresponds to a time of 2.04 seconds.
Note: For scaledown ramp_timeout:
- PrbDelta = 2b11 => 781.25 us
- PrbDelta = 2b10 => 1562.5 us
- PrbDelta = 2b01 => 3125 us
- PrbDelta = 2b00 => 6250 us
