SD_CDn_CTRL (IOU_SLCR) Register

SD_CDn_CTRL (IOU_SLCR) Register Description

Register NameSD_CDn_CTRL
Relative Address0x000000035C
Absolute Address 0x00FF18035C (IOU_SLCR)
Width32
TypemixedMixed types. See bit-field details.
Reset Value0x00000000
DescriptionSDIO Card Detect (CDn) connection:

0: Connect CDn input signal to MIO or EMIO device pin; used with removable SD/MMC card. 1: Connect CDn to force assertion of CDn; the card is assumed to be present.
