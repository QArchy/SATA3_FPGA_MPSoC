PCTRL (SATA_AHCI_VENDOR) Register

PCTRL (SATA_AHCI_VENDOR) Register Description

Register NamePCTRL
Relative Address0x0000000000
Absolute Address 0x00FD0C00A0 (SATA_AHCI_VENDOR)
Width32
TypemixedMixed types. See bit-field details.
Reset Value0x00000000
DescriptionSerDes control AHB control port.

Port SERDES Control. The SERDES is designed with an AHB control port. For integration simplicity, Vendor provides a facility within the Vendor-SATA Controller IP to control this SERDES AHB bus via simple reads/writes to a defined address within the Vendor IP.
