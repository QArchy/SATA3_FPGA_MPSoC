VCC_PSPLL (AMS) Register

VCC_PSPLL (AMS) Register Description

Register NameVCC_PSPLL
Relative Address0x0000000060
Absolute Address 0x00FFA50060 (AMS_CTRL)
Width32
TyperoRead-only
Reset Value0x00000000
DescriptionSystem PLLs voltage measurement, VCC_PSPLL.

Includes all five PS system PLLs: IOPLL, RPLL, APLL, DPLL, and VPLL. Does not include the six DDR I/O PLLs. (register was called vcc_pspll0)
